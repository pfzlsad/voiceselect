基于ModelScope的声纹特征提取与相似度计算系统，支持多目标说话人识别和音频片段自动切分。

## 📋 项目概述

本项目实现了完整的声纹识别工作流程，包括音频预处理、精细切分、多目标声纹比对和结果整理。系统能够从长音频中识别出目标说话人的片段，适用于语音分析、音频监控、内容审核等场景。

## ✨ 主要特性

- **智能音频预处理**：自动检测并转换采样率至16kHz，支持多种音频格式
- **精细音频切分**：基于声学特征（能量、过零率、频谱质心）的智能分段
- **多目标声纹比对**：支持多个目标说话人样本，提供多种相似度融合策略
- **完整工作流程**：端到端的处理流程，从原始音频到结果报告的自动化处理
- **详细分析报告**：生成文本和JSON格式的详细分析报告
